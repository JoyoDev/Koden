# Koden (WIP)

<img src="https://github.com/JoyoDev/Koden/raw/main/scripts/logo.png" width="200">

----

Koden is a simple orchestration tool for Docker containers. 
Similar to other orchestration systems, Koden includes the following
components:

- The task
- The scheduler
- The manager
- The worker
- The cluster
- The CLI
